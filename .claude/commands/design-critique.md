# Landing Page Design Critique

Comprehensive design evaluation using modern UX/UI principles and best practices.

## Evaluation Criteria:

### Visual Hierarchy
- Information architecture clarity
- Eye path analysis (F-pattern/Z-pattern)
- Typography scale and readability
- Color contrast and harmony
- Spacing and layout balance

### User Experience
- Navigation intuitiveness
- Mobile responsiveness
- Load perception optimization
- Error state handling
- Accessibility compliance (WCAG 2.1)

### Conversion Design
- Value proposition visibility
- Trust signals placement
- Friction point identification
- Form optimization
- Social proof integration

### Brand Consistency
- Visual identity alignment
- Tone of voice consistency
- Imagery style coherence
- Component system usage

## Critique Process:

1. **First Impression Analysis** (5-second test)
2. **Detailed Component Review**
3. **User Flow Evaluation**
4. **Competitive Comparison**
5. **Recommendation Prioritization**

## Deliverable:
Structured critique document with:
- Strengths identification
- Critical issues to fix
- Improvement opportunities
- Priority ranking
- Specific implementation suggestions
