# Generate Optimized Landing Page

Create a new, high-converting landing page using industry best practices and modern web technologies.

## Parameters:
- **industry**: Target industry (e.g., SaaS, e-commerce, consulting)
- **goal**: Primary objective (e.g., lead generation, sales, sign-ups)

## Generation Process:

### 1. Research & Strategy
```bash
# Research industry standards
web search: "[industry] landing page best practices 2025"

# Get competitive insights
skill: "seo-expert" analyze="[industry] keywords"
```

### 2. Component Generation
```bash
# Generate base components
skill: "next.js-page-generator" type="landing-page"

# Get design system
skill: "frontend-design" industry="[industry]"

# Fetch UI components
mcp shadcn search_items_in_registries registries='["@shadcn"]' query="hero section pricing"
```

### 3. Code Implementation
- Next.js 15 with App Router
- Tailwind CSS + Shadcn/ui
- TypeScript for type safety
- Framer Motion for animations
- SEO-optimized metadata

### 4. Performance Optimization
- Next.js Image optimization
- Dynamic imports for code splitting
- Font optimization
- Critical CSS inlining

### 5. Conversion Features
- Multiple CTA placements
- Social proof sections
- FAQ accordion
- Trust badges
- Testimonial carousel
- Pricing comparison table

## Output:
Complete landing page codebase with:
- Responsive design (mobile-first)
- SEO optimization
- Performance optimizations
- Conversion tracking ready
- Accessibility compliant
- Modern animations
