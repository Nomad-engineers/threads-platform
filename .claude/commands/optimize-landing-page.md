# Landing Page Optimization Workflow

Execute a comprehensive landing page optimization using all available tools and agents.

## Execution Steps:

1. **Initial Analysis Phase**
   - Activate Landing Page Agent for conversion analysis
   - Run SEO Expert audit
   - Perform Chrome DevTools performance trace
   - Use sequential-thinking for strategic analysis

2. **Planning Phase**
   - Create optimization roadmap with Planner-Manager
   - Set specific KPIs and success metrics
   - Prioritize fixes by impact/effort matrix

3. **Implementation Phase**
   - Engage Senior Next.js Agent for technical optimizations
   - Use Shadcn/ui for modern UI components
   - Implement design system with Frontend Design skill
   - Apply SEO best practices

4. **Testing Phase**
   - Cross-browser testing with Chrome DevTools
   - Performance validation
   - SEO verification
   - A/B testing setup

5. **Launch & Monitor**
   - Deploy with confidence
   - Set up monitoring dashboard
   - Create continuous optimization loop

## Parameters:
- `--analyze`: Run comprehensive analysis only
- `--implement`: Skip analysis, implement optimizations
- `--test`: Run testing and validation only
- `--monitor`: Set up monitoring and tracking

## Expected Output:
Fully optimized landing page with improved conversion rates, page speed, SEO rankings, and user engagement metrics.
